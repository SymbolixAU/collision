# Wedge-tailed Eagle data

A non-stochastic example of bird data for vignettes and testing Based
off the Australian / Tasmanian Wedge-tailed Eagle

## Usage

``` r
wte
```

## Format

A object of class 'list' and custom class 'birdInput'

## Details

Created with
[`define_bird()`](https://symbolixau.github.io/collision/reference/define_bird.md)
