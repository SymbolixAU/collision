# Rotate areas

Rotate shape through 180 degrees. Internal function

## Usage

``` r
rotate_areas(front, side)
```

## Arguments

- front:

  area of front-view in metres squared

- side:

  area of side-view in metres squared

## Value

numeric

## Examples

``` r
if (FALSE) { # \dontrun{
rotate_areas(1,2)
} # }
```
