# Distance model example - raptor

An object of class `dsmodel` fit using
[`Distance::ds`](https://rdrr.io/pkg/Distance/man/ds.html) The example
model is fit to the data `df_obs_survey` from this package

## Usage

``` r
ds_raptor
```

## Format

An object of class `dsmodel` of length 3.
